# Structure and Interpretation of Computer Programs

Repository for Lisp Code from the book *Structure and Interpretation of Computer Programs*.

* * *

## How to Run Lisp

``` bash
sbcl --script filename.lisp
```

* * *

## Supported file types for Common Lisp

- .lisp
- .lsp
- .l
- .cl
- .asd
- .asdf
